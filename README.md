# AgriWorld Media

## Project Overview
AgriWorld Media is a comprehensive platform that brings together the latest news, expert insights, and a vibrant community forum focused on agriculture and agribusiness. Users can find professional resources and engage with the community to share knowledge and experiences.

## Features
- **Latest News**: Stay updated with the latest happenings in the agricultural sector.
- **Expert Insights**: Gain valuable knowledge from industry experts.
- **Community Forum**: Engage with like-minded individuals and share ideas.
- **Professional Resources**: Access a plethora of resources to aid your agricultural ventures.
- **Responsive Design**: Enjoy a seamless experience across devices.

## Live Website
Visit our live website at [AgriWorld Media](https://Evans50.github.io/agriworld-media/).

## Setup Instructions for Local Development
1. Clone the repository: `git clone https://github.com/Evans50/agriworld-media.git`
2. Navigate to the project directory: `cd agriworld-media`
3. Open the project in your favorite code editor.

## Deployment Guides
### GitHub Pages
- GitHub Pages is already enabled. To deploy, simply push changes to the `main` branch.

### Netlify
1. Sign in to Netlify.
2. Click on 'New site from Git'.
3. Select GitHub and authorize your account.
4. Choose the `agriworld-media` repository for deployment.
5. Configure the build settings as necessary.

### Vercel
1. Sign up for Vercel.
2. Import your GitHub repository.
3. Configure your settings and deploy.

### Firebase Hosting
1. Install the Firebase CLI: `npm install -g firebase-tools`
2. Initialize Firebase in your project: `firebase init`
3. Deploy to Firebase Hosting: `firebase deploy`

- **Adding Images**: Place images in the `assets/images/` folder and link them in your HTML.
- **Adding Pages**: Create new HTML files and link them in the navigation.
